# 漫画阅读器产品介绍与功能按钮总览

> 文档版本：对应主程序 `0.7.3.1`
> 文档用途：面向产品梳理、交互核对、后续 UI 重构与功能验收。
> 说明：本文按当前 Windows 主线版本整理，macOS 端暂不纳入功能验收范围。

## 1. 产品定位

本项目是一款离线优先的 Windows 本地漫画书库管理与阅读软件。它不提供在线漫画源，不抓取远程元数据，不做云同步；核心目标是把用户本地漫画文件夹整理成可管理、可筛选、可持续维护的个人漫画库，并提供足够清晰、稳定、低干扰的阅读器。

当前产品关键词：

- 本地文件夹驱动：每个直接包含图片的文件夹识别为一本漫画。
- 离线元数据管理：标题、作者、标签、简介、封面、阅读进度、收藏、隐私状态都保存在本地 SQLite。
- 高价值数据安全：数据库支持备份、恢复、数据目录迁移；删除库记录不删除硬盘漫画源文件。
- 阅读体验优先：阅读器支持单页/双页、适宽/适高/原始、质量/性能渲染模式、目录、下一本、快捷键和沉浸控制。
- 视觉统一：按钮、标签、弹窗、详情页、书库卡片统一使用圆角矩形与主题 Token，避免碎片化 UI。

## 2. 功能框架总览

### 2.1 主界面框架

主界面由侧边栏、主页、书库、标签页、作者页、日志区、详情浮层和若干弹窗组成。

- 侧边栏：负责全局导航、数据安全、隐私模式、更新与日志入口。
- 主页：展示继续阅读、最近阅读等高频作品入口。
- 书库：展示完整漫画集合，支持筛选、排序、批量管理、Tag 池。
- 标签页：管理所有候选 Tag、分组、颜色、规则和关联作品。
- 作者页：管理作者、作者筛选、作者改名和删除。
- 详情浮层：展示单本漫画详情，并提供阅读、编辑、目录、危险操作。
- 阅读器窗口：独立窗口，负责看图、翻页、适配、渲染模式、目录和阅读状态保存。

### 2.2 数据与服务框架

- `LibraryScanner`：扫描书库根目录，边发现边扫描，上报进度，支持取消。
- `LibraryDatabase`：SQLite 数据库，保存书籍、作者、Tag、阅读进度、封面页、隐私状态、设置。
- `BatchImportAnalyzer`：分析作者文件夹批量导入候选。
- `ImageLoader`：阅读器图片加载与解码策略。
- `CoverCache` / `CoverThumbnailPipeline`：封面缓存与缩略图生成。
- `TagCatalog` / `TagService`：Tag 分类、颜色、规则与候选项管理。
- `UpdateService`：检查更新、下载或定位更新包、启动更新器。
- `MangaReader.Updater`：替换旧版本文件并重启主程序。
- `AppStorage`：应用数据目录、备份目录、日志目录管理。
- `AppLogger`：运行日志记录。

## 3. 侧边栏按钮与全局入口

### 3.1 一级导航按钮

- `主页`：切换到主页视图，展示继续阅读和最近阅读卡片。
- `书库`：切换到完整漫画库视图，显示筛选栏、Tag 池、瀑布流列表。
- `标签`：切换到标签管理页，用于创建、编辑、删除、筛选 Tag。
- `作者`：切换到作者管理页，用于新增、筛选、改名、删除作者。

### 3.2 工具区按钮

- `设置`：打开设置面板，包含通用（隐私模式、删除源文件密码）、阅读（快捷键、滚轮模式、渲染质量、双页间距）、数据（数据根目录、备份恢复、检查更新）、标签（自定义颜色）、危险（清除书签、重置设置）五个分区。
- `检查更新`：主动检查新版本。当前更新链路支持本地发布目录、本地 zip 包和 GitHub latest Release 兜底。
- `展开日志 / 收起日志`：显示或隐藏底部日志面板。

## 4. 主页功能与按钮

主页用于低成本回到最近阅读或继续阅读的作品。

- `导入漫画`：当书库为空或需要添加书库路径时，进入导入流程。
- `去漫画库`：从主页跳转到书库视图。
- 作品卡片点击：打开该作品详情浮层，不直接进入阅读器。
- `查看详情`：作品卡片内按钮，显式打开详情浮层。

主页卡片交互：

- 卡片有圆角、边框和 hover 反馈。
- 继续阅读区域保持等宽布局，避免三张作品宽度不一致。
- 隐私模式开启或单书隐私封面开启时，卡片封面显示遮罩。

## 5. 书库页功能与按钮

### 5.1 顶部筛选与排序

- `专注浏览`：隐藏或收起书库上方的筛选/辅助区域，让瀑布流获得更大显示空间。
- 搜索框：按标题、作者、Tag、简介、出品时间、阅读状态等字段进行文本检索。
- `状态：全部 / 状态：未读 / 状态：在读`：按阅读状态筛选作品。
- `只看收藏`：只显示已收藏作品。
- `排序：标题`：按标题排序。
- `排序：进度`：按阅读进度排序。
- `排序：页数`：按总页数排序。
- `排序：容量`：按图片总容量排序。
- `排序：阅读次数`：按阅读次数排序。
- `排序：录入时间`：按加入数据库时间排序。
- `排序：出品时间`：按用户维护的出品时间排序。
- `升序 / 降序`：切换当前排序字段的方向。标题默认升序；进度、页数、容量、阅读次数、录入时间、出品时间默认降序，用户可手动反向。
- `清空筛选`：清除搜索、状态、作者、收藏、隐藏、Tag 等筛选条件。
- `显示隐藏作品`：将已隐藏作品纳入当前书库列表。

### 5.2 多选管理区

- `多选管理`：进入批量选择模式，卡片显示选择框。
- `全选`：选择当前可见列表中的全部作品。
- `清空`：清除当前批量选择。
- `去前缀`：批量移除选中作品标题中的指定前缀。
- `样式 1 / 样式 2 / 样式 3 / 样式 4`：选择要批量应用的卡片样式。
- `应用`：把当前选择的卡片样式应用到选中作品。
- `+ Tag`：给选中作品批量添加 Tag。优先使用已有 Tag；空输入时会给已有 Tag 候选；输入不存在的名称才进入新建 Tag 流程。
- `- Tag`：从选中作品批量移除 Tag。
- `退出`：退出批量选择模式。

### 5.3 Tag 池与筛选按钮

- `添加`：创建或加入候选 Tag，并用于筛选/管理。
- Tag 块单击：按该 Tag 筛选书库。
- Tag 块双击：进入书库筛选语义，适用于书库和详情页等多个场景。
- 筛选态 Tag 不套用“互斥 Tag 清同组”规则；同组多个 Tag 可以同时保留为筛选条件。
- Tag 右键菜单：
  - `编辑标签`：打开 Tag 编辑弹窗。
  - `删除标签`：删除候选 Tag。
  - `前往标签页`：跳转到标签管理页。

### 5.4 书库卡片交互

- 卡片单击：选中并打开详情浮层。
- 卡片双击：打开详情或进入阅读，具体行为由当前交互上下文控制。
- 卡片 Tag：支持 hover/press 反馈，点击可筛选，避免误触打开详情。
- 批量选择框：多选模式下勾选或取消选中作品。
- 隐私遮罩：全局隐私模式或单书隐私封面开启时覆盖真实封面。

## 6. 标签管理页功能与按钮

标签页用于维护候选 Tag，而不是在作品编辑页临时创建无管理归属的 Tag。

### 6.1 顶部入口

- `新建候选标签`：打开新建 Tag 弹窗。
- `去书库`：返回书库视图。

### 6.2 排序与筛选

- `按组排序`：按 Tag 分组聚合排序。
- `按使用量`：按关联作品数量排序。
- `按名称`：按 Tag 名称排序。
- `全部分组`：筛选指定 Tag 分组。

### 6.3 标签列表按钮

- `按此标签筛选`：跳转书库，并按当前 Tag 筛选作品。
- `编辑标签`：打开编辑 Tag 弹窗，可修改名称、分组、颜色和互斥规则。
- `删除`：删除当前候选 Tag。

### 6.4 标签右键菜单

- `编辑标签`：同列表按钮。
- `删除标签`：删除候选 Tag。
- `按此标签筛选`：进入书库筛选。

### 6.5 Tag 数据规则

- Tag 支持分组。
- 同组 Tag 使用统一颜色。
- Tag 可设置为互斥或不互斥。
- 作品编辑页只能从已有 Tag 中选择，不负责创建新 Tag。

## 7. 作者管理页功能与按钮

作者页用于集中维护作者名称和作者筛选，避免作品编辑页直接创建新作者。

- `新增作者`：创建一个暂未关联书籍的独立作者。
- `去书库`：返回书库视图。
- `按照此作者筛选`：进入书库，并按该作者筛选作品。
- `修改作者信息`：重命名作者；若该作者有关联书籍，会同步更新书籍作者字段。
- `删除`：删除作者记录。对有关联作品的作者，删除前需要明确确认。

作者列表会合并两类来源：

- 已有关联作品的作者。
- 用户手动创建但暂未关联作品的独立作者。

## 8. 作品详情浮层功能与按钮

详情浮层是单本作品的核心操作中心，分为展示态和编辑态。

### 8.1 顶部按钮

- `← 返回书库`：关闭详情浮层，回到书库瀑布流。
- `☆ 收藏 / ★ 已收藏`：切换收藏状态，并立即写入数据库。
- `编辑`：进入作品信息编辑态。
- `更多 ▾`：打开更多操作菜单。

### 8.2 更多菜单

- `打开所在文件夹`：在文件资源管理器中打开漫画源文件夹。
- `重新定位`：当文件夹移动后，重新绑定该作品路径。
- `切换卡片样式`：切换该作品在书库瀑布流中的卡片样式；详情页封面区会同步预览装帧效果。
- `隐藏作品 / 取消隐藏`：控制该作品是否在默认书库列表中隐藏。
- `删除库记录`：从数据库删除该作品记录，但不删除硬盘漫画源文件。

### 8.3 封面区按钮

- `打开目录`：在详情页内打开目录浮层，浏览页面缩略图。

### 8.4 Hero 区与阅读操作

- 作者按钮：点击后按该作者筛选书库。
- `开始阅读 / 继续阅读`：打开阅读器窗口，从当前进度或第一页进入。
- `-1`：阅读次数减一。
- `+1`：阅读次数加一。
- `打开所在文件夹`：同更多菜单入口。
- `重新定位`：同更多菜单入口。
- `隐藏作品 / 取消隐藏`：同更多菜单入口。

### 8.5 详情展示内容

- 阅读状态：展示未读/在读等状态和阅读次数。
- 简介：展示用户维护的作品简介。
- 作品信息：以 3 行 2 列显示作者、外文名、页数、封面页、出品时间、录入时间等信息。
- Tag：以统一圆角矩形 Tag 块显示，支持筛选交互。

### 8.6 编辑态按钮

- `取消`：退出编辑态，不保存当前修改。
- `保存信息`：保存标题、外文名、作者、简介、出品时间、Tag 等信息。
- `今日`：将出品时间或对应日期字段快速设为当天。
- `Tag 搜索`：在已有 Tag 中检索可添加项，编辑页不负责创建新 Tag。
- `已选 Tag`：展示当前作品已经绑定的 Tag，点击 Tag 上的 `×` 可移除。
- `可添加 Tag`：展示搜索命中的已有候选 Tag，点击即可加入作品。
- `保持隐私封面 / 取消隐私封面`：设置该作品是否始终遮挡封面。该状态独立于全局隐私模式。
- `隐藏作品 / 取消隐藏`：编辑态中的隐藏开关。
- `删除库记录`：编辑态中的危险删除入口。

编辑态布局规则：

- `危险操作` 与 `取消 / 保存信息` 同行排列，减少底部独立按钮行造成的高度跳变。
- Tag 编辑区为一行式横向容器：左侧是 `Tag 搜索 + 已选 Tag`，右侧是 `可添加 Tag`。
- 编辑态不再强制最小高度，目标是与详情展示态保持接近尺寸。

编辑态已经移除：

- 阅读次数编辑。
- 封面页编辑。
- 阅读状态编辑。

这些状态改由阅读器、目录设封面、详情页快捷操作等更明确的入口维护。

## 9. 详情页目录预览浮层

详情页目录用于在不打开阅读器的情况下查看页缩略图。

- 标题格式：`目录预览 · 作品标题`。
- `关闭`：关闭目录浮层，返回详情页。
- 点击遮罩空白处：关闭目录预览，返回详情页。
- 缩略图左键：预览该页面。
- 缩略图右键菜单 `设为封面`：把该页设置为作品封面。

布局规则：

- 左侧目录列表固定为四列缩略图。
- 右侧预览区占用剩余空间，优先保证预览页显示面积。
- 标题下方不再显示说明小字，减少顶部噪声。
- 内容面板内部点击不会关闭浮层，只有点外侧空白才返回。

性能策略：

- 首次只加载首批缩略图。
- 后续缩略图通过 hover 懒加载。
- 预览加载带请求序号和取消检查，避免旧预览覆盖新预览。

## 10. 阅读器窗口功能与按钮

阅读器是独立窗口，专注于图片显示和翻页。

### 10.1 下一本确认层

- `留在当前页`：停留在最后一页，不跳转下一本。
- `前往下一本`：按当前书库筛选和排序上下文打开下一本漫画。

### 10.2 阅读器目录

- `目录`：打开阅读器目录。
- `关闭`：关闭目录浮层。
- 缩略图右键菜单 `设为封面`：把对应页面设置为封面。

### 10.3 显示比例按钮

- `适宽`：图片按当前视口宽度适配。
- `适高`：图片按当前视口高度适配。
- `原始`：按原始物理像素语义显示，避免系统二次插值造成画质判断偏差。

### 10.4 渲染策略按钮

- `质量 / 性能`：切换阅读器渲染模式。
  - 质量模式：原图解码，适宽/适高下后台生成清晰适配图并轻量锐化。
  - 性能模式：保留更低开销的解码和显示路径。

### 10.5 阅读辅助按钮

- `背景`：切换阅读器背景色。
- `隐藏 / 显示`：隐藏或显示阅读器工具栏和控件。
- `全屏 / 窗口`：切换全屏模式；不再承担隐藏 UI 的状态循环。

### 10.6 下拉控件

- `滚轮翻页`：滚轮用于上一页/下一页。
- `滚轮缩放`：滚轮用于缩放。
- `滚轮滚动`：滚轮用于滚动画布。
- `单页`：单页阅读。
- `双页`：双页阅读。
- `从左到右`：普通阅读方向。
- `从右到左`：日漫式阅读方向。

### 10.7 阅读器快捷键语义

当前阅读器保留一套固定高频快捷键语义：

- 上一页 / 下一页：键盘或鼠标触发翻页。
- `W`：全屏。
- `E`：适应高度。
- `Q`：适应宽度。
- `D`：隐藏或显示 UI。
- `S`：单页 / 双页切换。
- `Z`：切换滚轮模式。
- `X`：退出阅读器。
- `C`：切换左右阅读顺序。
- `A`：切换背景色。

## 11. 导入相关弹窗

### 11.1 导入路径弹窗

- `×`：关闭弹窗。
- `加入路径`：添加一个书库或导入路径。
- `取消`：放弃导入。
- `继续导入`：确认路径并进入导入/扫描流程。

### 11.2 作者批量导入弹窗

- `×`：关闭弹窗。
- `取消`：放弃本次作者文件夹导入。
- `确认导入`：按当前候选列表和作者名执行批量导入。

作者批量导入能力：

- 可拖入作者文件夹。
- 递归识别内部直接包含图片的漫画目录。
- 导入前可确认作者名和每本漫画标题。
- 同一路径以更新处理，不重复创建。

## 12. Tag 弹窗

### 12.1 新建 Tag 弹窗

- `×`：关闭弹窗。
- `互斥 / 不互斥`：设置该 Tag 所属规则。
- `取消`：放弃创建。
- `创建`：保存新 Tag。

可配置项：

- Tag 名称。
- Tag 分组。
- Tag 颜色。
- 互斥规则。

### 12.2 编辑 Tag 弹窗

- `×`：关闭弹窗。
- `查看关联的漫画 →`：跳转查看使用该 Tag 的作品。
- `互斥 / 不互斥`：修改互斥规则。
- `取消`：放弃修改。
- `保存`：保存修改。

当前颜色规则：

- 分组颜色统一。
- 切换分组不再强制覆盖用户已经手动选择的颜色，避免误改。

### 12.3 Tag 名称弹窗

- `取消`：关闭，不保存。
- `保存`：确认名称输入。

## 13. 作者与重命名弹窗

### 13.1 通用重命名弹窗

- `取消`：放弃重命名。
- `确认`：保存新名称。

使用场景：

- 作者改名。
- 其他简单名称修改。

## 14. 数据安全面板

数据安全面板集中处理数据库安全和低频维护动作。

- `×`：关闭面板。
- `立即备份`：立刻复制当前数据库到备份目录。
- `恢复选中`：将选中的备份恢复为当前数据库。
- `打开备份目录`：打开 `backups` 文件夹。
- `打开数据目录`：打开当前应用数据目录。
- `选择其他 DB`：选择外部数据库文件作为恢复来源。
- `检查更新`：从数据安全面板触发更新检查。

安全约束：

- 恢复数据库前应确认当前数据库已有备份。
- 删除库记录不删除漫画源文件。
- 扫描、导入、更新不能覆盖用户手工维护的作者、Tag、简介等信息。

## 15. 日志面板

- `扩大 / 还原`：切换日志面板高度。
- `清空`：清空当前日志显示。
- `打开日志目录`：打开本地日志文件夹。

日志面板用于观察扫描、导入、更新、异常和调试信息。

## 16. 隐私功能

### 16.1 全局隐私模式

入口：侧边栏 `隐私模式：关 / 隐私模式：开`。

效果：

- 主页卡片封面遮挡。
- 书库卡片封面遮挡。
- 详情页大封面遮挡。
- 状态持久化，重启后保持。
- 只做视觉遮挡，不删除封面缓存。

### 16.2 单书隐私封面

入口：详情编辑态危险操作 `保持隐私封面 / 取消隐私封面`。

效果：

- 即使全局隐私模式关闭，该书仍保持封面遮挡。
- 数据库存储在 `books.is_privacy_cover`。
- 与全局隐私模式取并集：任一条件为真就显示遮罩。

## 17. 更新与发布框架

### 17.1 应用内更新

入口：

- 侧边栏 `检查更新`。
- 数据安全面板 `检查更新`。

更新顺序：

1. 检查本地 `_release` 中更高版本发布目录。
2. 检查本地更高版本 zip 更新包。
3. 检查 GitHub latest Release。

更新器行为：

- 支持目录或 zip 作为更新源。
- 跳过 `MangaReader_Data` 与数据目录指针文件，避免覆盖用户数据。
- 更新器先复制到临时目录运行，避免安装目录内 Updater 被锁。
- 更新完成后尝试自动重启主程序。
- 失败时写入临时目录错误日志。

### 17.2 发布硬规则

每次上新版本必须完成：

1. 更新主程序版本号。
2. 更新开发文档。
3. 本地 `dotnet build` 验证。
4. 本地 `dotnet publish -c Release -r win-x64 --self-contained true`。
5. zip 打包验证。
6. 验证发布目录和 zip 内均包含主程序与 `Updater/MangaReader.Updater.exe`。
7. 提交并推送 `main`。
8. 推送 `v*` tag 触发 GitHub Release workflow。

## 18. 当前功能边界

当前明确不做或暂缓：

- 不支持压缩包直接阅读。
- 不做在线漫画源。
- 不做云同步。
- 不抓取远程元数据。
- macOS 端暂时搁置。
- 编辑作品信息页不创建新作者和新 Tag。
- 阅读状态、阅读次数、封面页不再由作品编辑页直接维护。

## 19. 后续验收建议

后续每次 UI 或功能重构，应至少检查以下按钮链路：

- 侧边栏：主页、书库、标签、作者、指定数据、备份数据、隐私模式、检查更新、日志。
- 主页：查看详情、去漫画库、导入漫画。
- 书库：筛选、排序方向、多选管理、批量 Tag、Tag 筛选、隐藏作品显示。
- 标签页：新建、编辑、删除、按标签筛选、分组筛选。
- 作者页：新增、筛选、改名、删除。
- 详情页：返回、收藏、编辑、更多菜单、开始阅读、目录预览、阅读次数、隐私封面、危险操作。
- 阅读器：适宽、适高、原始、质量/性能、背景、目录、隐藏、全屏、单/双页、阅读方向、滚轮模式。
- 数据安全：备份、恢复、打开目录、选择 DB、检查更新。
