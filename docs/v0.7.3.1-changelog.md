# v0.7.3.1 更新日志

## 修复

### 目录间距恢复正常

阅读器目录（PageCatalogList）和详情目录（DetailCatalogList）的卡片间距此前被错误替换为 `VirtualizingWrapPanel` + `ScrollViewer.CanContentScroll="True"`，导致布局与原始 `WrapPanel` 行为不一致。

**修复方式**：将两个目录恢复为 `<WrapPanel/>`，移除 `CanContentScroll`，Border 加回原始 Margin：

| 位置 | Panel | Border Margin |
|------|-------|:---:|
| ReaderWindow PageCatalogList | `<WrapPanel/>` | `Margin="7"` |
| MainWindow DetailCatalogList | `<WrapPanel/>` | `Margin="6"` |

### 筛选 CheckBox 不再换色

"只看收藏"和"只看隐藏作品"两个 CheckBox 在勾选时背景色从 `Brush.SurfaceMuted` 变为 `Brush.TextPrimary`（深色），造成整条按钮变色。这与筛选打勾的预期行为不符。

**修复方式**：修改全局 CheckBox 样式的 `IsChecked` 触发器，移除 `CheckRoot` 的背景/边框/前景色变更，仅在小勾选框内显示勾号。

## 涉及文件

- `MangaReader.Native/MainWindow.xaml` — 详情目录 Panel/CheckBox IsChecked 触发器
- `MangaReader.Native/ReaderWindow.xaml` — 阅读器目录 Panel
- `MangaReader.Native/MangaReader.Native.csproj` — 版本号
